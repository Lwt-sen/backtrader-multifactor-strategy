# backtrader-multifactor-strategy
A股月度多因子量化回测策略（Backtrader实现）
## 项目简介
基于4个基本面因子横截面标准化打分选股：
1. 盈利因子：ROE（权重40%）
2. 成长因子：营收同比tr_yoy（权重25%）
3. 估值因子：PE、PB（合计权重35%）
### 核心机制
- 每月横截面Z-score标准化，消除时间/量纲偏差
- merge_asof 对齐财报与日线，全程无未来函数
- ATR波动率动态风控计算单笔买入仓位
- 月度调仓：剔除不在当月选股池的持仓，换入新高分标的
- 自动过滤ST股、上市不满1年次新股
